Fluid Simulation

Messages and attributes common to Fluid Simulation operators

Description

Arguments

None.

Messages

bang Computes fluids forces and colors and output the result matrix.
blob index [int]
x [float32]
y [float32]
Notify the new position of the blob identified by its index.
blob_apply_force index [int]
apply [int]
Determine if force (push) have to be applied on particles for the blob identified by its index.
blob_force index [int]
force [float32]
Set the force (push) of the blob identified by its index.
blob_info index [int]
Output some informations about the blob identified by its index (fourth outlet).
blob_list Output all existing index of blobs (fourth outlet).
blob_reset Remove all blobs.
blob_seuil index [int]
min [float32]
max [float32]
Set the threshold of every blobs' mouvements. Mouvements under the min value will be considered null, and mouvements upper the max value will be replaced by this value.
color Set a fixed color.
deltaT Set the deltaT value of the fluid solver. See more on MSAFluidSolver2D's doc.
fade_speed Set the fadeSpeed value of the fluid solver. See more on MSAFluidSolver2D's doc.
fluid_dim Set the dimension of the fluid solver. See more on MSAFluidSolver2D's doc.
iterations Set the number of iterations of the fluid solver. See more on MSAFluidSolver2D's doc.
no_color Enable or Disable the use of color (forces are always used). 0 = color, 1 = no color
random_color Enable or Disable the use of a random color. 0 = fixed, 1 = random
reset Reset the system. All values will be set as default.
velocity Set the velocity value of the fluid solver. See more on MSAFluidSolver2D's doc.
viscosity Set the viscosity value of the fluid solver. See more on MSAFluidSolver2D's doc.
x_scale min [float32]
max [float32]
Set the range of x values for input blob's coordinates.
y_scale min [float32]
max [float32]
Set the range of y values for input blob's coordinates.
